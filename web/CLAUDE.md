# TAG IT Network Website

## Project Overview
Blockchain-powered product authentication platform website with cyber-industrial minimalist aesthetic.

## Tech Stack
- Static HTML/CSS/JS (single-page app)
- Deployed on Vercel
- No build step required

## Design System
Colors:
- Background: #0A0E14 (Deep Carbon)
- Cards: #1A1F29 (Carbon Gray)
- Primary: #00D4FF (Electric Blue)
- Success: #00FF88 (Circuit Green)
- Secondary: #B84FFF (Neon Purple)
- Text: #F5F7FA (Neutral White)

Typography:
- Headings: Inter 700
- Body: Inter 400/500
- Code: JetBrains Mono

## Code Style
- Use CSS variables for all colors
- Mobile-first responsive design
- Keep animations GPU-accelerated (transform, opacity)
- Maintain overflow-x: hidden for mobile

## File Structure
- index.html - Complete single-page website
- vercel.json - Deployment config
- README.md - Project info
- TODO.md - Remaining tasks

## Key Sections
Navigation, Hero (3D cubes), Trust Stats, Features, Life Cycle, Solutions, Technology, Tokenomics, About, Footer
